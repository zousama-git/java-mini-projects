# Age and Student Status Checker

## Overview
This program collects a user’s name and age, categorizes their age group, and verifies student status.

## Concepts Covered
* User input handling with `Scanner`
* Conditional logic (`if-else if-else`) for categorization
* Validation of input
* Looping (`while`) for repeated user prompts until valid input
* String comparison using `equalsIgnoreCase`
